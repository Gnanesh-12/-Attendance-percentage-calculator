# 🎓 Attendance Percentage Calculator

![Project Banner](Screenshot%202025-05-06%20120003.png)

A sleek, modern, and highly intuitive web application designed for students to manage their attendance with precision. Whether you need to catch up on classes or are planning your next "bunk" day, this tool provides the exact data you need.

---

## ✨ Features

- **🎯 Precision Calculation:** Calculate exactly how many more classes you need to attend or how many you can safely skip.
- **📊 Dynamic Visualization:** Beautifully rendered doughnut charts using **Chart.js** for a clear breakdown of present vs. absent days.
- **🎨 Premium UI/UX:** A modern aesthetic featuring:
  - **Glassmorphism** effects.
  - Smooth **Micro-animations** for a premium feel.
  - **Responsive Design** that works flawlessly on mobile and desktop.
  - Curated typography using the **Outfit** font from Google Fonts.
- **💡 Real-time Insights:** Instant feedback on your attendance status relative to your target percentage.

---

## 🚀 Tech Stack

- **HTML5:** Semantic structure for optimal accessibility.
- **CSS3:** Custom styles with modern techniques (CSS Variables, Flexbox, Grid, Animations).
- **JavaScript (ES6+):** Core logic for attendance calculations and dynamic interactions.
- **Chart.js:** Robust library for high-quality data visualization.
- **Google Fonts:** Utilizing the *Outfit* typeface for a clean, modern look.

---

## 🛠️ How to Use

1. **Select Target:** Choose your desired attendance percentage (e.g., 75%, 85%).
2. **Enter Data:** Input your current "Days Present" and "Total Classes".
3. **Calculate:** Hit the "Calculate Now" button.
4. **Analyze:** View your personalized attendance breakdown and actionable advice.
5. **Visualize:** Check the doughnut chart to see your current standing at a glance.

---

## 📦 Installation & Setup

No complex setup required! This is a client-side web application.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gnanesh-12/-Attendance-percentage-calculator.git
   ```
2. **Open the project:**
   Navigate to the project folder and open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
